## 플래닛랜드에 오신것을 환영합니다!
플래닛랜드는 웹으로 즐기는 간단한 매매(우주ver) 게임입니다. <br>
행성을 사고 팔며 돈을 모아 다른 여러 행성을 소유할 수 있고 <br>
업그레이드를 통해 더 많은 돈을 얻을 수 있습니다. 
<br>

## 개발 환경
- 운영체제: macOS <br>
- IDE: IntelliJ <br>
- Java 8 <br>
- Spring boot 2.7.8 <br>
- Spring Web <br>
- Spring Data JPA <br>
- Spring Security <br>
- 데이터베이스: MySQL <br>
- Bootsrap, Thymeleaf <br>
